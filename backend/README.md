# HimShakti Backend API

## How to Run Backend Locally

### 1. Open Terminal

Navigate to the backend folder:

```bash
cd backend
```

### 2. Install Dependencies

```bash
npm install
```

### 3. Create Environment File

Create a `.env` file and add:

```
PORT=5000
```

### 4. Start the Server

```bash
npm run dev
```

### 5. Open API

Visit:

```
http://localhost:5000/api/products
```

The backend server will start on port 5000 and provide REST API endpoints for the HimShakti application.
